# **App Name**: CertifyYT

## Core Features:

- YouTube Link Input: Accepts YouTube video or playlist URLs from users.
- MCQ Generation: Generates MCQs from YouTube video titles using the Gemini API; uses a tool to determine whether to regenerate questions for the same playlist.
- MCQ Test Display: Displays a test with 20 random MCQs.
- Score Calculation & Result: Calculates the test score and determines pass/fail status based on a 76% threshold.
- Certificate Generation: Generates a certificate with the user's name, video title, certificate ID, and a QR code after passing the test.
- Certificate Download: Provides a "Download Certificate" button for JPG download.
- Certificate Validation: A page to validate certificates via a unique QR code/certificate ID.

## Style Guidelines:

- Primary color: Slate Blue (#7A8B99) to inspire trust and competence.
- Background color: Very light grey (#F0F2F5), of a similar hue to the primary.
- Accent color: Sky Blue (#87CEEB) to signal important elements and interactivity.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple and clear icons from a consistent set, emphasizing comprehensibility over ornamentation.
- Employ a clean and spacious layout that adheres to common web design patterns. Mobile-optimized.
- Use subtle animations and transitions to enhance user experience (e.g., loading states, button hovers).
